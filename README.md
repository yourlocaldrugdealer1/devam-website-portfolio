# Devam Soni — Web Design Portfolio

A portfolio and catalogue site for freelance web design work, built for local businesses in Ahmedabad — cafes, salons, and small shops that don't have a website yet.

**Live site:** https://yourlocaldrugdealer1.github.io/devam-website-portfolio

## What's in this repo

- `index.html` — the main portfolio site, including a catalogue of sample work, the design process, pricing, and contact details.
- `kasturi-cafe-template.html` — a fully built sample website for a fictional Ahmedabad café, used to demonstrate the kind of site clients can expect.

## About this project

Most small local businesses in Ahmedabad rely entirely on Google Maps listings, Justdial, or Instagram for their online presence — with no actual website of their own. This portfolio is built to pitch a simple alternative: a clean, fast, mobile-friendly one-page site that gives customers a real place to see the menu or services, photos, hours, and location, all in one spot.

The Kasturi café template is the first example: a single-page site with a menu section styled like a chalkboard, a story section, a photo gallery, and contact details — built to show what a finished project looks like before a client commits to anything.

## Tech

Plain HTML, CSS, and a small amount of vanilla JavaScript (for scroll animations) — no frameworks, no build step, no dependencies. Hosted free on GitHub Pages.
